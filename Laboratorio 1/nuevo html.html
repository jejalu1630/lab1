El diseño web implica trabajo relacionado con el layout y diseño de páginas online, así como la producción de contenido, aunque generalmente se aplica a la creación de sitios web. En este caso, los diseñadores web crean las páginas utilizando lenguajes de marcado como HTML o XML

El desarrollo de una página web es un proceso que involucra una serie de fases interdependientes, desde la planificación y diseño inicial hasta la programación, pruebas y lanzamiento. Cada etapa es crucial para garantizar que el sitio web sea atractivo, funcional y eficiente

Imagen tomada de google

Un sitio web es un espacio en la red donde puedes almacenar información para compartirla con los demás. Puede contene información sobre ti mismo, tu negocio o incluso temas de tu interés. Según la categoría del sitio web, la gente puede utilizarlo para comprar, chatear, estudiar o entretenerse

Las partes de una pagina web son:

  1. Encabezado
  2. Menu
  3. Imágenes
  4. Página de inicio
  5. Contenido del sitio web
  6. Pie de pagina
